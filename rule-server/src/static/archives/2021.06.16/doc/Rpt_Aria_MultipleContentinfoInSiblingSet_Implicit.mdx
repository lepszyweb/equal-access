### Znaleziono na stronie wiele elementów z rolą `"contentinfo"`

<div id="locLevel"></div>

Strona, dokument lub aplikacja może mieć tylko jeden element z rolą `"contentinfo"`.

</Column>

### Dlaczego to jest ważne

Na stronie (w dokumencie, aplikacji) powinien istnieć tylko jeden element z rolą `"contentinfo"`, aby uniknąć zamieszania co do tego, gdzie znaleźć informacje o treści. Element z rolą `"contentinfo"` jest punktem orientacyjnym, który identyfikuje typowe informacje stopki o jego rodzicu, takie jak prawa autorskie, dostępność i oświadczenia o prywatności, zwykle powtarzane na dole każdej strony. Element HTML `<footer>` ma niejawną rolę `"contentinfo"`, gdy jego kontekstem jest element `<body>`.

<div id="locSnippet"></div>

### Co zrobić

* Usuń rolę `"contentinfo"` ze wszystkich elementów poza jednym, który reprezentuje stopkę dla głównej treści  strony, dokumentu lub aplikacji.

Na przykład:

<CodeSnippet type="multi" light={true}>&lt;body&gt;
	&lt;div role="main"&gt;Treść główna...&lt;/div&gt;
	&lt;div role="search"&gt;Wyszukiwarka&lt;/div&gt;
	&lt;div role="contentinfo"&gt;Informacje o prawach autorskich...&lt;/div&gt;
&lt;/body&gt;</CodeSnippet>
