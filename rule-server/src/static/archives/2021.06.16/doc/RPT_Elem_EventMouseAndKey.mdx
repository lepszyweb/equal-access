### Sprawdź, czy treść interaktywna z obsługą zdarzeń myszy ma równoważny dostęp z klawiatury

Cała treść interaktywna z obsługą zdarzeń myszy musi mieć równoważny dostęp z klawiatury

### Dlaczego to jest ważne

Skrypty obsługi zdarzeń specyficzne dla myszy mogą być używane do wyzwalania działań kontrolek graficznego interfejsu użytkownika (GUI). Równoważne programy obsługi specyficzne dla klawiatury umożliwiają osobom, które polegają na klawiaturze lub interfejsie klawiatury, dostęp do tych samych funkcji.


<div id="locSnippet"></div>

### Co zrobić

* Używając tylko klawiatury, upewnij się, że możesz uzyskać dostęp do wszystkich funkcji zapewnianych przez obsługę zdarzeń myszy.
* ORAZ, jeśli nie ma równoważnego dostępu z klawiatury, postępuj zgodnie z poniższą tabelą obsługi zdarzeń, aby zapewnić odpowiednie obsługi zdarzeń klawiatury:


| Użyj          | z          |
| :------------ | :------------ |
| onmousedown   | onkeydown     |
| onmouseup     | onkeyup       |
| onclick       | onkeypress    |
| onmouseover   | onfocus 		|
| onmouseout    | onblur        |

</Column>
