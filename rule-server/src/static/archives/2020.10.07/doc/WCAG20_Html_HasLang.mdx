### Strona HTML nie identyfikuje swojego domyślnego języka za pomocą atrybutu `lang`.

<div id="locLevel"></div>

Strona musi identyfikować domyślny język dokumentu za pomocą atrybutu `lang`.

</Column>
</Row>

### Dlaczego to jest ważne?

Syntezatory mowy, które obsługują wiele języków, mogą dostosować się do specyficznej dla danego języka wymowy i składni, mówiąc tekst we właściwym języku z właściwą wymową. Niektóre czytniki ekranu mogą być skonfigurowane do automatycznego przełączania języków w oparciu o atrybut `lang`.


<div id="locSnippet"></div>

### Co zrobić

* Określ domyślny język dokumentu HTML poprzez podanie atrybutu `lang` w elemencie HTML (np. `<html lang="pl">`);
* LUB, jeśli obsługuje tylko dokument XML (np. używając typu MIME takiego jak aplikacja/xhtml+xml), określ domyślny język poprzez podanie samego atrybutu `xml:lang`. Atrybut `lang` nie jest wymagany (`<html xml:lang="pl" xmlns="http://www.w3.org/1999/xhtml">`)
* LUB, jeśli jako tekst/html służy XHTML 1.x lub strony wielojęzyczne, użyj zarówno atrybutu `lang` jak i atrybutu `xml:lang` i upewnij się, że wartości dla obu atrybutów są identyczne (`<html lang="pl" xml:lang="pl" xmlns="http://www.w3.org/1999/xhtml">`)
