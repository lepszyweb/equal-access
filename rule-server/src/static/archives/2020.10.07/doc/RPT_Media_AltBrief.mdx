### Dlaczego to jest ważne

Wartości atrybutu `alt` ('tekst alt') są przeznaczone do krótkiego, liniowego zastąpienia obrazu, gdy nie można go załadować lub wyświetlić. Przegadany lub długi tekst alt może mieć negatywny wpływ na doświadczenie użytkownika.


<div id="locSnippet"></div>

### Co zrobić

* Użyj opisu `alt` o długości mniejszej niż 150 znaków;
* LUB, jeśli krótszy opis nie jest odpowiedni, użyj atrybutu `aria-describeby`, aby wskazać wartość `id` elementu zawierającego dłuższy opis;
* LUB użyj odnośnika 'D', jak pokazano w przykładzie.
