### Sprawdź, czy wartość `blink` właściwości CSS` text-decoration` nie jest używana dłużej niż pięć sekund

<div id="locLevel"></div>

Nie używaj wartości „blink” właściwości CSS „text-decoration” dłużej niż przez pięć sekund

</Column>

### Dlaczego to jest ważne

Migające treści mogą być bardzo rozpraszające, uniemożliwiając niektórym osobom skupienie się na innych treściach. Mruganie przez maksymalnie 5 sekund daje czas na zwrócenie uwagi użytkownika, ale jest wystarczająco krótkie, aby mógł on przeczekać rozproszenie uwagi przed przeczytaniem strony.


<div id="locSnippet"></div>

### Co zrobić

* Nie używaj wartości `blink` we właściwości CSS `text-decoration`;
* LUB zapewnij na stronie internetowej kontrolkę, która pozwoli użytkownikom zatrzymać miganie;
* LUB użyj skryptów, aby zatrzymać miganie po 5 sekundach.


</Column>
