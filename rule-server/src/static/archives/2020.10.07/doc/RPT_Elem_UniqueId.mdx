### Element `<{0}>` ma `id` \"{1}\", które nie jest unikalne

<div id="locLevel"></div>

Wartości atrybutu `id` elementów muszą być unikalne w ramach dokumentu

</Column>

### Dlaczego to jest ważne?

Atrybut `id` zapewnia elementom strony identyfikator do odczytu maszynowego. Unikalna wartość `id` umożliwia przeglądarkom i technologiom wspomagajacym dokładną interpretację i przetwarzanie treści.

<div id="locSnippet"></div>

### Co zrobić

* Każdemu elementowi, który jest identyfikowany atrybutem `id`, nadaj unikalną niepustą wartość atrybutu.

Na przykład:

<CodeSnippet type="multi" light={true}>&lt;h1 id="nowiny"&gt;Wiadomości codzienne&lt;/h1&gt;
&lt;p id="nowinyIntro"&gt;Najnowsze wiadomości z całego świata&lt;/p&gt;
...
&lt;div role="article" aria-labelledby="news" aria-describedby="newsintro"&gt;
    ...
&lt;/div&gt;</CodeSnippet>
