### Istnieje element z rolą `"contentinfo"`, ale nie ma elementu `"main"`.

<div id="locLevel"></div>

Element z rolą `"contentinfo"` jest dozwolony tylko, gdy istnieje element z rolą `"main"`

</Column>
</Row>
<Row>
<Column colLg={11} colMd={5} colSm={4} className="toolMain">

### Dlaczego to jest ważne

Element z rolą `"contentinfo"` jest obszarem, który określa wspólne informacje na dole każdej strony, zwykle nazywany stopką strony. Zawiera informacje takie jak prawa autorskie i łącza do oświadczeń o prywatności i dostępności. Stopka bez głównej treści w ramach roli `"main"` nie jest dozwolona. Element HTML `<footer>` ma niejawną rolę `"contentinfo"`, gdy jest bezpośrednim potomkiem elementu `<body>`.

<div id="locSnippet"></div>

### Co zrobić

* Dodaj element `<main>`, który ma niejawną rolę `"main"`, aby umieścić główną treść strony internetowej.

Na przykład:

<CodeSnippet type="multi" light={true}>&lt;body&gt;
    &lt;div role="main"&gt;
      ...
    &lt;/div&gt;
    &lt;div role="contentinfo"&gt;
	    ...
    &lt;/div&gt;
&lt;/body&gt;</CodeSnippet>
