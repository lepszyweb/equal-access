### Istnieje wiele elementów z rolą `"article"` bez unikalnych etykiet

<div id="locLevel"></div>

Każdy element z rolą `"article"` powinien mieć widoczną, niepowtarzalną etykietę

### Dlaczego to jest ważne

Element HTML `<article>`lub dowolny element z jawną rolą `"article"` jest typem obszaru kluczowego WAI-ARIA. Jeśli strona zawiera więcej niż jeden obszar artykułów, każdy z nich powinien mieć unikalną etykietę, aby osoby korzystające z technologii pomocniczych mogły szybko zidentyfikować i przejść do każdego obszaru `"article"`.


<div id="locSnippet"></div>

### Co zrobić

* Dla każdego elementu `<article>` lub z `role="article"` ustaw wartość atrybutu `aria-labelledby` na wartość `id` dowolnego widocznego tekstu, który odpowiednio i jednoznacznie opisuje treść tego obszaru;
* LUB użyj atrybutu `aria-label`, aby dodać niewidoczną, unikalną, niepustą etykietę do każdego elementu z rolą `"article"`.

Na przykład:

<CodeSnippet type="multi" light={true}>&lt;article aria-label="Prognoza pogody"&gt;
    ...
&lt;/article&gt;
    
&lt;div role="article" aria-labelledby="t1"&gt;
    &lt;h1 id="t1"&gt;Notowania giełdowe&lt;/h1"&gt;
    ...
&lt;/div&gt;</CodeSnippet>
