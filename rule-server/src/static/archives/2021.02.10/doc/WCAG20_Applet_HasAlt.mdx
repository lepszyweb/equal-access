### Element `<applet>` nie ma atrybutu `alt`, który zapewnia krótki tekst alternatywny

<div id="locLevel"></div>

Elementy `<applet>` muszą zawierać tekst `alt` i opis alternatywny

</Column>
</Row>
<Row>
<Column colLg={11} colMd={5} colSm={4} className="toolMain">

### Dlaczego to jest ważne

Tekst alternatywny w elementach `<applet>` jest ważny dla użytkowników, którzy nie mają przeglądarki obsługującej język Java lub mają wyłączoną obsługę języka Java™. Ponadto zapewnienie alternatywy tekstowej sprawia, że te same informacje są dostępne za pośrednictwem dźwięku lub innych kanałów.

<div id="locSnippet"></div>

### Co zrobić

* Jeśli element `<applet>` nie ma opisu alternatywnego, dodaj atrybut `alt`
* ORAZ dodaj treść oparta na HTML (tj. tekst wewnętrzny), która opisuje funkcję elementu `<applet>`, jeśli to możliwe. 

Na przykład:

<CodeSnippet type="multi" light={true}>&lt;applet CODE="banner.class" ALT="Przewijany baner wiadomości" WIDTH="200" HEIGHT="200"&gt;
Baner przedstawiający trzy rzeczy:
 &lt;ol&gt;
  &lt;li&gt;Średnia wartość Dow Jones Industrial wzrósła dziś o 300 punktów w ciężkich transakcjach.&lt;/li&gt;
  &lt;li&gt;Szczyt G8 rozpocznie się jutro w Waszyngtonie.&lt;/li&gt;
  &lt;li&gt;Globalny traktat o nierozprzestrzenianiu broni jądrowej podpisany dzisiaj przez wszystkie narody.&lt;/li&gt;
 &lt;/ol&gt;
&lt;/applet&gt;</CodeSnippet>
