### Dlaczego to jest ważne

Wizualne wygląd odstępów w tekście powinien być uzyskany przy użyciu arkuszy stylów, aby zachować sensowną sekwencję tekstu. Samo dodanie pustych znaków w celu kontroli odstępów może zmienić znaczenie, wymowę lub interpretację słowa lub spowodować, że słowo nie będzie programowo rozpoznawane jako pojedyncze słowo.

<div id="locSnippet"></div>

### Co zrobić

 * Sprawdź, czy znaki spacji nie są używane do tworzenia spacji między literami słowa;
 * LUB, jeśli między literami lub wyrazami muszą być zwiększone odstępy, [użyj właściwości CSS `letter-spacing`](https://www.w3.org/WAI/WCAG21/Techniques/css/C8.html).
