### Element `<figure>` nie ma przypisanej etykiety
<div id="locLevel"></div>

Element `<figure>` musi mieć przypisaną etykietę

</Column>

<Column colLg={11} colMd={5} colSm={4} className="toolMain">

### Dlaczego to jest ważne

Element `<figure>` jest używany zwykle do opatrywania adnotacjami ilustracji, diagramów, zdjęć, list kodów itp. Skojarzenie z etykietą sprawia, że te same informacje są dostępne poprzez audio lub inne kanały.

<div id="locSnippet"></div>

### Co zrobić

* Dodaj element `<figaption>` wewnątrz elementu `<figure>`, aby zapewnić widoczny podpis, który prezentuje treść elementu `<figure>`; 
* LUB dodaj do elementu `<figure>` atrybut `aria-labelledby`, którego wartość wskazuje na element z widocznym na ekranie tekstem, majacym znaczenie etykiety (podpisu);

Na przykład:

<CodeSnippet type="multi" light={true}>&lt;figure&gt;
	&lt;figcaption&gt;Zamek na przestrzeni wieków: odpowiednio 1423, 1858 i 1999.&lt;/figcaption&gt;
	&lt;figure&gt;
	&lt;figcaption&gt;Etching. Anonymous, ca. 1423.&lt;/figcaption&gt;
	&lt;img src="castle1423.jpeg" alt="Zamek ma jedną wieżę i wysoki mur wokół niej."&gt;
	&lt;/figure&gt;
&lt;/figure&gt;</CodeSnippet>
