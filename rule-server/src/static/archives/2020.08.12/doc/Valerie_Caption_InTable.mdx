### Element `<caption>` nie jest zagnieżdżony w elemencie `<table>`

<div id="locLevel"></div>

Element `<caption>` musi być umieszczony wewnątrz powiązanego elementu `<table>`

</Column>
</Row>
<Row>
<Column colLg={11} colMd={5} colSm={4} className="toolMain">

### Dlaczego to jest ważne?

Element `<caption>` jest identyfikatorem i działa jak tytuł lub nagłówek tabeli. Umożliwia czytnikom ekranu przejście bezpośrednio do napisu, aby uzyskać opis tabeli.

<div id="locSnippet"></div>

### Co zrobić

 * Umieść element `<caption>` wewnątrz elementu `<table>`.

 Na przykład:

<CodeSnippet type="multi" light={true}>&lt;table id="data_table1" border="1" summary="Prosta tabela danych przedstawiająca liczbę chłopców i dziewcząt w każdej klasie szkoły podstawowej"&gt;
        &lt;caption&gt;Dziewczęta i chłopcy w klasach szkół podstawowych&lt;/caption&gt;
        &lt;tr&gt; ... &lt;/tr&gt;
        &lt;tr&gt; ... &lt;/tr&gt;
&lt;/table&gt;</CodeSnippet>

</Column>
