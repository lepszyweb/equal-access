### Dlaczego to jest ważne?

Gdy znaczniki są używane do wizualnego formatowania tekstu jako listy, ale nie wskazuje na związek z listą, użytkownicy mogą mieć trudności ze zrozumieniem i poruszaniem się po informacjach.

<div id="locSnippet"></div>

### Co zrobić

 * Upewnij się, że treść nie jest listą;
 * LUB, jeśli jest to lista, użyj elementów listy HTML (np. `<ol>`, `<ul>`, `<li>`), aby utworzyć listę i CSS, aby wystylizować listę zgodnie z wymaganiami.
