### Dlaczego to jest ważne

Użyj CSS, aby dołączyć tylko obrazy dekoracyjne. Obrazy tła w CSS nie obsługują alternatywnych tekstów. Gdy użytkownicy nie widzą domyślnego tła strony, wszelkie ważne informacje z obrazów tła CSS są tracone.

<div id="locSnippet"></div>

### Co zrobić

* Jeśli obraz w tle zawiera ważny tekst, należy użyć prawdziwego tekstu, a nie obrazu tekstu;
* LUB, jeśli obraz tła przekazuje ważne informacje, użyj elementu `<img>` z atrybutem `"alt"` zamiast obrazu tła;
* LUB zapewnij tekst alternatywny dla informacji, które zostaną pokazane, gdy obraz CSS nie zostanie wyświetlony.

</Column>
