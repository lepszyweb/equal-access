### Dlaczego to jest ważne

Jeśli we `<frame>` znajduje się treść inna niż HTML, może być niedostępna. Na przykład, jeśli źródłem elementu `<frame>` jest plik obrazu, technologia wspomagająca nie może odczytać treści, ponieważ nie można dodać tekstu `alt` do obrazu w źródle `<frame>`. Użytkownik otwiera ramkę i nic nie słyszy albo słyszy nazwę pliku obrazu. Element `<frame>` jest przestarzały i nie powinien być używany. Kiedy jest używany, źródło ramki musi być dostępne.


<div id="locSnippet"></div>

### Co zrobić

 * Element HTML `<frame>` jest przestarzały, jeśli to możliwe, unikaj jego używania;
 * LUB  sprawdź, czy zawartość interaktywna wewnątrz ramki jest dostępna za pomocą klawiatury;
 * ORAZ sprawdź, czy dostęp do treści w ramce można uzyskać za pomocą technologii wspomagających, takich jak czytnik ekranu.
