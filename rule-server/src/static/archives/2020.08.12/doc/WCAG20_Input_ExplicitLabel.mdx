### Dlaczego to jest ważne

Powiązanie znaczącej etykiety z każdą kontrolą interfejsu użytkownika pozwala przeglądarce i technologii wspomagającej na ujawnienie i&nbsp;przekazanie użytkownikowi informacji o kontrolce. Powiązanie widocznej etykiety zapewnia również większy obszar, który można kliknąć.

<div id="locSnippet"></div>

### Co zrobić

* Dodaj atrybut `for` do widocznej etykiety z `id` tej kontrolki (np. `<label for="idKontrolki">`);
* LUB umieść element UI wewnątrz widocznego elementu `<label>`;
* LUB dodaj do kontrolki atrybut `aria-labelledby`. Musi on wskazywać na element z widocznym na stronie tekstem, który ma znaczenie jako etykieta;
* LUB tylko wtedy, gdy projekt nie może mieć widocznej etykiety, użyj atrybutu `title`, aby zapewnić etykietę.
