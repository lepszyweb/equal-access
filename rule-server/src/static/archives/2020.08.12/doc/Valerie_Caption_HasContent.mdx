<div id="locLevel"></div>

Element `<caption>` w elemencie `<table>` musi zawierać opisowy tekst

### Dlaczego to jest ważne?

Element `<caption>` jest identyfikatorem i działa jak tytuł lub nagłówek tabeli. Umożliwia czytnikom ekranu przejście bezpośrednio do napisu, aby uzyskać opis tabeli.

<div id="locSnippet"></div>

### Co zrobić

* Dodaj tekst opisowy do elementu `<caption>`.

Na przykład:

<CodeSnippet type="multi" light={true}>&lt;table id="data_table1" border="1"&gt;
&lt;caption&gt;Dziewczęta i chłopcy w klasach szkół podstawowych&lt;/caption&gt;
	&lt;tr&gt;
		&lt;th scope="col"&gt;Klasy&lt;/th&gt;
		&lt;th scope="col"&gt;# dziewcząt&lt;/th&gt;
		&lt;th scope="col"&gt;# chłopców&lt;/th&gt;
	&lt;/tr&gt;
&lt;/table&gt;</CodeSnippet>

</Column>
