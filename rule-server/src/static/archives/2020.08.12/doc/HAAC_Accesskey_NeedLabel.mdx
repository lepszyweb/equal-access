### Element HTML z przypisanym atrybutem `accesskey` nie ma przypisanej etykiety

### Dlaczego to jest ważne

Etykieta elementu HTML z atrybutem `accesskey` pozwala programowi użytkownika na wyświetlenie listy klawiszy dostępu z nazwą opisującą funkcję każdego z nich. Pozwala również użytkownikom korzystajacym z kontroli głosowej wypowiadać etykietę, aby aktywować jego funkcję.

<div id="locSnippet"></div>

### Co zrobić

* Podaj etykietę, używając atrybutu `title` (np. `<a title="Aktywności" accessskey="A" href="/konsorcjum/aktywnosci">Aktywności</a>`);
* lub, użyj `<label for="">` lub `aria-labelledby` do oznaczenia widocznego tekstu jako etykiety;
* lub, użyj wejścia wbudowanego w `<label>` (np. `<label><input type="checkbox" accesskey="A"/>foo</label>`);
* lub, jeśli element nie ma widocznej etykiety, podaj etykietę używając atrybutu `aria-label` (np. `aria-label="Aktywności"`).


</Column>

* Osoby niewidome korzystające z czytników ekranu
* Osoby, które nie mogą korzystać z urządzeń wskazujących, np. myszy
* Osoby o ograniczonej sprawności ruchowej korzystające ze sterowania za pomocą głosu
