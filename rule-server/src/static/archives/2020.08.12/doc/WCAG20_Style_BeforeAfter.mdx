### Zweryfikuj, czy pseudoelementy `::before` i `::after` nie wstawiają treści niedekoracyjnej

<div id="locLevel"></div>

Nie używaj pseudoelementów CSS `::before` i `::after` do wstawiania treści niedekoracyjnej.

</Column>

### Dlaczego to jest ważne

Pseudoelementy CSS `::before` i `::after` określają położenie treści, która ma być wstawiona przed lub za treścią elementu  w drzewie dokumentu. W przypadku użytkowników, którzy muszą dostosować lub wyłączyć informacje o stylu, aby mogli wyświetlać treść zgodnie ze swoimi potrzebami, technologie wspomagające mogą nie być w stanie uzyskać dostępu do informacji, które są wstawiane przy użyciu CSS.

<div id="locSnippet"></div>

### Co zrobić

 * Zweryfikuj, czy treść wstawiona przez pseudoelementy CSS `::before` i `::after` jest czysto dekoracyjna;
 * LUB wyłącz style CSS i sprawdź, czy niedekoracyjne informacje we wstawionej treści nadal istnieją (są widoczne, można je odczytać);
 * LUB umieść ważne informacje bezpośrednio na stronie.

### Na kogo to wpływa

 * Osoby z wadami wzroku korzystające z przełączników kontrastu kolorów
 * Ludzie, którzy używają alternatywnego tła, aby zwiększyć czytelność
 * Ludzie, którzy dostosowują kolory swojego wyświetlacza, aby ułatwić sobie czytanie
 * Osoby używające tylko tekstu, wyświetlaczy monochromatycznych lub brajlowskich
 * Osoby korzystające z przeglądarek tekstowych (np. Lynx) lub interfejsów audio
