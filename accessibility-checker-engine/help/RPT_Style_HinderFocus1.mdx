### Sprawdź, czy wskaźnik fokusu klawiatury jest dobrze widoczny, gdy używasz elementów CSS dla obramowania lub konturu


<div id="locLevel"></div>

Wskaźnik fokusu klawiatury musi być dobrze widoczny, gdy domyślne obramowanie lub kontur są modyfikowane przez CSS.

</Column>
</Row>
<Row>
<Column colLg={11} colMd={5} colSm={4} className="toolMain">

### Dlaczego to jest ważne

Wszystkie wskaźniki fokusu klawiatury powinny być dobrze widoczne i rozpoznawalne. Podczas nawigacji po stronie internetowej za pomocą klawiatury użytkownicy muszą wiedzieć, który element ma fokus klawiatury. Wizualnym wskaźnikiem fokusa może być pionowy pasek (typowy kursor) lub widoczne obramowanie otaczające element sterujący interfejsu użytkownika. 



<div id="locSnippet"></div>

### Co zrobić

 * Jeśli fokus klawiatury spełnia standardy widoczności podczas przechodzenia do komponentu klawiszem Tab, nie jest wymagane żadne działanie;
 * LUB użyj domyślnego wskaźnika fokusu dla platformy;
 * LUB dostosuj styl CSS, aby fokus klawiatury był dobrze widoczny;
 * LUB użyj komponentów interfejsu, które są podświetlane przez klienta użytkownika, gdy otrzymują fokus;
 * LUB użyj skryptu, aby zmienić kolor tła lub obramowanie elementu, aby był wizualnie wyraźny, gdy otrzyma fokus
