### Dlaczego to jest ważne

Tryb wysokiego kontrastu systemu Windows zmienia sposób wyświetlania treści internetowych, wyłączając niektóre funkcje CSS. Aplikacje, które używają obrazów tła CSS dla dowolnego komponentu mogą mieć problemy, takie jak znikanie obrazów lub elementów interfejsu użytkownika, gdy tryb wysokiego kontrastu systemu Windows jest włączony. Innym przykładem jest zniknięcie wizualnych wskaźników fokusa i wizualne zmiany elementów interfejsu użytkownika przy włączonym trybie wysokiego kontrastu systemu Windows.


<div id="locSnippet"></div>

### Co zrobić

 * Na komputerze z systemem Windows aktywuj tryb wysokiego kontrastu;
 * ORAZ potwierdź, że wszystkie elementy interfejsu użytkownika są widoczne i dziedziczą wybrany motyw wysokiego kontrastu;
 * ORAZ potwierdź, że pola wyboru, przyciski opcji i granice pól wprowadzania tekstu są widoczne z dobrym kontrastem;
 * ORAZ potwierdź, że wizualny wskaźnik fokusu jest dobrze widoczny;
 * ORAZ potwierdź, że wszystkie obrazy przekazujące informacje są widoczne lub wyświetlana jest alternatywa tekstowa;
 * ORAZ potwierdź, że wszystkie obrazy dekoracyjne nie są już widoczne.
